Harry Potter quiz

Harry Potter-themed quiz application with questions, answer options, result messages, audio feedback, DOM manipulation, and key features like event handling, quiz flow control, CSS styling, and object-oriented programming. It showcases asynchronous JavaScript, dynamic DOM manipulation, event delegation, and audio preloading for an engaging quiz experience.

Tic Tac Toe

Tic Tac Toe game with the option to play against another player or the computer. The computer opponent places O randomly and lacks advanced intelligence. Features include sound effects, visually appealing CSS design, and logic for checking winning combinations.

Mario from Wish

This JavaScript code introduces a Mario-themed game implemented on an HTML canvas. The player is controlled by keyboard keys, featuring animations for running and jumping. The canvas rendering showcases Mario, various platforms, and a pyramid-shaped obstacle. Users can generate a pyramid with a customizable height, resembling a specific pattern. The "Player" class oversees Mario's actions, including velocity, animation, and collision detection with platforms and the pyramid.

Jokes

The code employs the generateJoke function to retrieve Chuck Norris or dad jokes from APIs. It dynamically modifies the content of the "jokeText" HTML element, responding to the selected joke type. This process involves asynchronous operations, DOM manipulation, and API requests within the code.
